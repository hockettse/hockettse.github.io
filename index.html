Spencer Cox and Sarah Hockett

List of 'static' pages

We are using handlebar protocols, so below is a description of each page

  • Home: The main home page of the site. Displays a welcome screen and links to the subpages (stars, planets, constellations, galaxies
  • Stars: Allows a user to add a star or look up information on stars.
  • Constellations: Allows a user to look up information on constellations.
  • Planets: Allows a user to add a planet or look up information on planets.
  • Planetary Systems: Allows a user to add a planetary system or look up information on planetary systems.
  • Galaxy: Allows a user to add a galaxy or look up information on galaxies.
